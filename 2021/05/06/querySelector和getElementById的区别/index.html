querySelector 和 getElementById 的区别

getElementsByTagName() 速度比 querySelectorAll() 快的根本原因在于动态 NodeList 和静态 NodeList 对象的不同。

总结 (摘抄自网上的)

  • getElementsByTagName() 速度比 querySelectorAll() 快的根本原因在于动态 NodeList 和静态 NodeList 对象的不同。
  • 哪个方法更好用主要还是看你的需求，如果只是要根据 tag name 来查找元素，也不需要获取此一个快照，那就应该使用 getElementsByTagName() 方法；如果需要快照结果（静态）, 或者需要使用复杂的 CSS 查询，则可以考虑 querySelectorAll()。
