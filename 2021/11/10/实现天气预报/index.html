用 Java 实现天气预报

起因

  • 因每次去上班都不知道今天下不下雨。要是下雨鞋子就会湿的。
  • 然后逛 github 发现有一个 QQ 机器人。可以用 Java 调用。遂想到实时预报下雨的天气。
  • 所以设置每天早上 6 点发消息提醒今天的降水概率。

Step

  1. 使用 simbot QQ 机器人 (github 开源的）
  2. 使用和风天气和阿里云的易源数据作为天气 API
